## How to work on this repository

- You first need to assign yourself a issue.
- Then you can create a branch through the issue and GitHub will give you the necessary commands to change to your branch on terminal.
- Then you can git commit and git push your chances and create a pull request to merge your branch to main branch.
    - At least one person needs to aprove your PR for you to be able to merge it.
 
### Running the code 

Run this command to build your code.
- You will need to have installed docker compose
  
```
docker compose up -d --build
```
